# 지게차운전기능사 필기 모의고사

이밤새tv 회차별 기출복원 자막을 바탕으로 만든 지게차운전기능사 필기 모의고사입니다.

## 구성
- `index.html` — 메인 페이지 (모든 회차로 연결)
- `round1.html` ~ `round9.html` — 1~9회 모의고사 (각 60문제, 즉시 채점)
- `freq.html` — 1~9회 540문항 빈출 총정리

## 특징
- 보기 선택 시 즉시 정답·해설 표시
- 맞은/틀린 개수 및 합격 여부(36문제 이상) 실시간 표시
- 법규 문항은 2026년 기준으로 검증
